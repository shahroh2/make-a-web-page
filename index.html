Introduction to web programming

Lesson1:The Basics of the Web and HTML

World wide web

Web:The major pieces of web are computer, browsers, the internet and servers. It is a primarily collection of HTML documents.

HTML:It is main document type on web and glues everything together on web.It stands for hypertext markup language and is made up of:-
  • text content(what you see)
  • markups(how it looks like)
  • references to other documents and links to other pages(image/videos)

Urls: It is how you refer to documents on web.

HTTP:It is the mail protocol of web and used by internet/browsers to communicate with servers.

HTML markup basics
HTML markups are made of Tags and Elements:
  • Tag:a anchor tag, b bold tag,em emphasis tag,p paragraph tag,br break tag, img image(img src=”url” alt=”text”.It is a void tag with no contents)
  • Elements: (opening tag) contents of tag (closing tag)
  • Attributes: (open tag -> attr=”value”) contents (close tag) Ex: This website is my favorite
Note: Inline vs Block elements.The block elements create invisible boxes around contents inside.
  • Inline:br/span/a/img/strong tags
  • Block:p/div tags

Lesson2:Creating a Structured Document with HTML

3 languages’ for web development
  • HTML:Hypertext markup language is main document of web. It provides Structure for webpage.
  • CSS:A framework which helps develop webpage quickly. It provides Style for webpage.
  • JS:Javascript are interactive components of webpage.

Note:Google chrome->Tools->Developer tool shows webpage structure.

Tree structure of browser

The elements are structured inside each other which can contain elements or contents.Programmers often talk about DOM(Document Object Model) which is tree-like structure of a page and tandard convention for interacting in HTML.

HMTL is all about boxes!

Everything on webpage is actually a box. Once we define box structure for webpage then all the elements can be easily re-arranged around those boxes to create different user experience.

3 Code editors’ tools and their utility

Lesson3:Adding CSS style to HTML structure

Cascading Style sheets

CSS stands for Cascading style sheets. Programmers should know that avoiding repetition is essential to all areas of programming. We use CSS to avoid that repetition. You can have several CSS files/sheets which is used to find out invisible style for webpage.

The two concepts to understand with CSS are:-
  • Inheritance: properties application to not only a specified element but also its descendants.
  • Abstract thinking: applying generality amongst seemingly different things”.
CSS is basically a search and replace tool.You identify a class or tag of element you want to find and then what you want to do it or what property values to replace with different ones. This feature of CSS has powerful advantage and disadvantage:-
  • Advantage: Using CSS one can avoid repetition in our code lets us save time and avoid creating errors due to manual addition of similar code.
  • Disadvantage: If a programmer ends up changing CSS and has errors it will cascade to many sections of code.
